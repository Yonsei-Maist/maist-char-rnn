# Char-RNN

module of character-based RNN

## About the Project

### Summary
 - train and predict something using RNN based on Character-level
 - support English
 - support Korean
 - support both of English and Korean (if mixed)

### Functions
 - Train and Test Char-RNN
 - Predict something from characters
